== Description ==

Shows the date & time of the last scheduled post in the publish box. Also comes with a shortcode [next-scheduled-post] to show the next schedlued post in your theme.

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the `Plugins` screen in WordPress
3. go edit an already published post
